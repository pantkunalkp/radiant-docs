# Radio

A radio, when clicked or tapped triggers an action or a series of actions.

Below are the relevant attributes for all radio types.

### Sizes

Radios are available in 2 sizes: `sm`, `md` (default) and `md`

The hierarchy for text size in radio labels is:

- **Body1** for medium radios
- **Body2** for small radios

```jsx live
function RadioSizes() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <Radio size="sm" color="primary" label="This is radio" />
          <Radio size="md" color="primary" label="This is radio" />
        </StyledBox>
      </CssVarsProvider>
    </div>
  );
}
```

### Variants

Radios are available in 4 global variants: `solid` , `soft`, `outlined`(default) and `plain`

```jsx live
function RadioVariants() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <Radio size="sm" variant="outlined" />
          <Radio size="md" variant="outlined" />
        </StyledBox>
      </CssVarsProvider>
    </div>
  );
}
```

### Colors

All components do support 6 color type: `primary`, `neutral`, `danger`, `info`, `success` and `warning`

```jsx live
function RadioVariants() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <Radio size="md" color="primary" defaultChecked />
          <Radio size="md" color="danger" defaultChecked />
          <Radio size="md" color="neutral" defaultChecked />
          <Radio size="md" color="success" defaultChecked />
          <Radio size="md" color="info" defaultChecked />
          <Radio size="md" color="warning" defaultChecked />
        </StyledBox>
      </CssVarsProvider>
    </div>
  );
}
```

### Disabled

Radios in disabled mode-

```jsx live
function RadioVariants() {
  return (
    <div>
      <CssVarsProvider>
        <StyledBox>
          <Radio size="md" disabled />
          <Radio size="md" defaultChecked disabled />
          <Radio
            size="md"
            label="This is disabled radio"
            defaultChecked
            disabled
          />
        </StyledBox>
      </CssVarsProvider>
    </div>
  );
}
```
