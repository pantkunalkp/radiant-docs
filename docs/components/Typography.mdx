# Typography

Use typography to present your design and content as clearly and efficiently as possible.

Below are the relevant attributes for all typography types.

### Sizes

The hierarchy for text size in typography - h1 > h2 > h3 > h4 > h5 > h6 > subtitle1 > subtitle2 > body1 > body2

```jsx live
function TypographySizes() {
  return (
    <Box>
      <Typography level="h1">H1</Typography>
      <Typography level="h2">H2</Typography>
      <Typography level="h3">H3</Typography>
      <Typography level="h4">H4</Typography>
      <Typography level="h5">H5</Typography>
      <Typography level="h6">H6</Typography>
      <Typography level="subtitle1">Subtitle1</Typography>
      <Typography level="subtitle2">Subtitle2</Typography>
      <Typography level="body1">Body1</Typography>
      <Typography level="body2">Body2</Typography>
    </Box>
  );
}
```

### Variants

Radios are available in 4 global variants: `solid`(default) , `soft`, `outlined` and `plain`

```jsx live
function TypographyVariants() {
  return (
    <StyledBox>
      <Typography variant="solid" level="h4">
        H4
      </Typography>
      <Typography variant="soft" level="h4">
        H4
      </Typography>
      <Typography variant="outlined" level="h4">
        H4
      </Typography>
      <Typography variant="plain" level="h4">
        H4
      </Typography>
    </StyledBox>
  );
}
```

### Colors

All components do support 6 color type: `primary`, `neutral`, `danger`, `info`, `success` and `warning`

```jsx live
function TypographyVariants() {
  return (
    <StyledBox>
      <Typography color="primary" level="h4">
        H4
      </Typography>
      <Typography color="neutral" level="h4">
        H4
      </Typography>
      <Typography color="danger" level="h4">
        H4
      </Typography>
      <Typography color="info" level="h4">
        H4
      </Typography>
      <Typography color="success" level="h4">
        H4
      </Typography>
      <Typography color="warning" level="h4">
        H4
      </Typography>
    </StyledBox>
  );
}
```
