### ğŸˆ Usage

1. Import CssVarsProvider and wrap the main `App` component with it.

```jsx
import { CssVarsProvider } from "@intugine-technologies/radiant";

ReactDOM.render(
  <CssVarsProvider>
    <App />
  </CssVarsProvider>
)
```

2. Import the Poppins font into root file.

```jsx
    # Only import the following fontweight since we only be using below fontweights

    import '@fontsource/poppins/400.css'
    import '@fontsource/poppins/500.css'
    import '@fontsource/poppins/600.css'
    import '@fontsource/poppins/700.css'
```

3. Import the component and use it like any other React component.

```jsx
import Button from '@intugine-technologies/radiant/Button'

<Button>Click Me!</Button>
```
