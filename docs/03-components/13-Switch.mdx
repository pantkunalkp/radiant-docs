# Switch

A switch, when clicked or tapped triggers an action or a series of actions.

Below are the relevant attributes for all switch types.

### Sizes

Switches are available in 2 sizes: `sm`, `md` (default) and `md`

```jsx live
function SwitchSizes() {
  return (
    <StyledBox>
      <Switch size="sm" />
      <Switch size="md" />
    </StyledBox>
  );
}
```

### Variants

Switches are available in 4 global variants: `solid` (default), `soft`, `outlined` and `plain`

```jsx live
function SwitchVariants() {
  return (
    <StyledBox>
      <Switch variant="solid" defaultChecked={true} />
      <Switch variant="soft" defaultChecked={true} />
      <Switch variant="outlined" defaultChecked={true} />
      <Switch variant="plain" defaultChecked={true} />
    </StyledBox>
  );
}
```

### Colors

All components do support 6 color type: `primary`, `neutral`, `danger`, `info`, `success` and `warning`

```jsx live
function SwitchVariants() {
  return (
    <StyledBox>
      <Switch variant="solid" color="primary" />
      <Switch variant="solid" color="neutral" />
      <Switch variant="solid" color="danger" />
      <Switch variant="solid" color="info" />
      <Switch variant="solid" color="success" />
      <Switch variant="solid" color="warning" />
    </StyledBox>
  );
}
```
