# Breadcrumbs

Breadcrumbs consist of a list of links that help a user visualize a page's location within the hierarchical structure of a website, and allow navigation up to any of its "ancestors".
Below are the relevant attributes for all breadcrumbs types.

```jsx live
function BreadcrumbSizes() {
  return (
    <Box>
      <Breadcrumbs>
        <Link href="#">User</Link>
        <Link href="#">Kunal</Link>
        <Link href="#">Office</Link>
        <Link href="#">Package</Link>
        <Typography>Radiant</Typography>
      </Breadcrumbs>
    </Box>
  );
}
```

### Separator

Breadcrumbs Separator - You can pass any

```jsx live
function BreadcrumbVariants() {
  return (
    <Box>
      <Breadcrumbs separator={<ChipDelete variant="plain" />}>
        <Link href="#">User</Link>
        <Link href="#">Kunal</Link>
        <Link href="#">Office</Link>
        <Link href="#">Package</Link>
        <Typography>Radiant</Typography>
      </Breadcrumbs>
      <Breadcrumbs separator=">">
        <Link href="#">User</Link>
        <Link href="#">Kunal</Link>
        <Link href="#">Office</Link>
        <Link href="#">Package</Link>
        <Typography>Radiant</Typography>
      </Breadcrumbs>
    </Box>
  );
}
```
