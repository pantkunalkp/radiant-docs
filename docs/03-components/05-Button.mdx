# Button

A button, when clicked or tapped triggers an action or a series of actions.

Below are the relevant attributes for all button types.

### Sizes

Buttons are available in 3 sizes: `sm`, `md` (default) and `lg`

The hierarchy for text size in button labels is:

- **H4** for large buttons
- **H5** for medium buttons
- **H6** for small buttons

```jsx live
function ButtonSizes() {
  return (
    <StyledBox>
      <Button size="lg">Large</Button>
      <Button size="md">Medium</Button>
      <Button size="sm">Small</Button>
    </StyledBox>
  );
}
```

### Variants

Buttons are available in 4 global variants: `solid` (default), `soft`, `outlined` and `plain`

```jsx live
function ButtonVariants() {
  return (
    <StyledBox>
      <Button variant="solid">Solid</Button>
      <Button variant="soft">Soft</Button>
      <Button variant="outlined">Outlined</Button>
      <Button variant="plain">Plain</Button>
    </StyledBox>
  );
}
```

### Buttons with icons

Buttons do have startDecorator and endDecorator props

```jsx live
function ButtonVariants() {
  return (
    <StyledBox>
      <Button variant="outlined" startDecorator={<DockIcon />}>
        This is Button
      </Button>
      <Button variant="solid" endDecorator={<DockIcon />}>
        This is Button
      </Button>
      <Button
        variant="soft"
        startDecorator={<DockIcon />}
        endDecorator={<DockIcon />}
      >
        This is Button
      </Button>
      <Button
        variant="plain"
        startDecorator={<DockIcon />}
        endDecorator={<DockIcon />}
      >
        This is Button
      </Button>
    </StyledBox>
  );
}
```

### Colors

All components do support 6 color type: `primary` (default), `neutral`, `danger`, `info`, `success` and `warning`

```jsx live
function ButtonVariants() {
  return (
    <StyledBox>
      <Button color="primary">primary</Button>
      <Button color="neutral">neutral</Button>
      <Button color="danger">danger</Button>
      <Button color="info">info</Button>
      <Button color="success">success</Button>
      <Button color="warning">warning</Button>
    </StyledBox>
  );
}
```

### IconButton

Use the `IconButton` component if you want width and height to be the same while not having a label.
Every prop previously covered are available for this component as well.

```jsx live
function ButtonSizes() {
  return (
    <StyledBox>
      <IconButton>
        <DockIcon />
      </IconButton>
      <IconButton>
        <TruckMovingIcon />
      </IconButton>
      <IconButton>
        <EpodIcon />
      </IconButton>
    </StyledBox>
  );
}
```
