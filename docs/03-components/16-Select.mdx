# Select

Select components are used for collecting user provided information from a list of options. 

## Introduction

The Select component is used to trigger a popup that displays a list of Option components.

## Component

Below are the relevant attributes for all Select types.

### Sizes

Select is available in 3 sizes: `sm`, `md` (default) and `lg`

The hierarchy for text size in select labels is:

- **Body2** for small radios
- **Body1** for medium and large radios

```jsx live
function selectSizes() {
  return (
    <StyledBox>
      <Select size="sm" value={10}>
        <Option value={10}>10</Option>
        <Option value={50}>50</Option>
        <Option value={100}>100</Option>
      </Select>
      <Select size="md" value={10}>
        <Option value={10}>10</Option>
        <Option value={50}>50</Option>
        <Option value={100}>100</Option>
      </Select>
      <Select size="lg" value={10}>
        <Option value={10}>10</Option>
        <Option value={50}>50</Option>
        <Option value={100}>100</Option>
      </Select>
    </StyledBox>
  );
}
```
