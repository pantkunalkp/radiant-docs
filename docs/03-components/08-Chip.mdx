# Chip

A Chip, when clicked or tapped (if clickable) triggers an action or a series of actions.

Below are the relevant attributes for all chip types.

### Sizes

Chips are available in 2 sizes: `sm`, `md` (default) and `lg`

```jsx live
function ChipSizes() {
  return (
    <StyledBox>
      <Chip size="sm">small</Chip>
      <Chip size="md">medium</Chip>
      <Chip size="lg">large</Chip>
    </StyledBox>
  );
}
```

### Colors

All components do support 6 color type: `primary`, `neutral`, `danger`, `info`, `success` and `warning`

```jsx live
function ChipVariants() {
  return (
    <StyledBox>
      <Chip variant="outlined" color="primary">Primary</Chip>
      <Chip variant="outlined" color="neutral">Neutral</Chip>
      <Chip variant="outlined" color="danger">Danger</Chip>
      <Chip variant="outlined" color="info">Info</Chip>
      <Chip variant="outlined" color="success">Success</Chip>
      <Chip variant="outlined" color="warning">Warning</Chip>
    </StyledBox>
  );
}
```

### Delete Button

To add a delete action inside a chip, use the complementary `ChipDelete` component. 
Note that its design will automatically adapt to the parent `Chip`.


```jsx live
function DeleteButtonChip() {
  return (
    <StyledBox>
      <Chip variant="outlined" color="danger" endDecorator={<ChipDelete />}>
        Remove
      </Chip>
      <Chip
        variant="soft"
        color="danger"
        endDecorator={<ChipDelete variant="plain" />}
      >
        Delete
      </Chip>
    </StyledBox>
  );
}
```

### Disabled

Chips in disabled mode

```jsx live
function ChipVariants() {
  return (
    <StyledBox>
      <Chip disabled>Chip</Chip>
      <Chip
        disabled
        endDecorator={<ChipDelete onClick={() => alert("Hi There")} />}
      >
        Chip
      </Chip>
    </StyledBox>
  );
}
```
