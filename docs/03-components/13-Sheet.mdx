# Sheet

he background of an application resembles the flat, opaque texture of a sheet of paper, and an application's behavior mimics paper's ability to be re-sized, shuffled, and bound together in multiple sheets.

Below are the relevant attributes for all sheet types.

```jsx live
function SheetSizes() {
  return (
    <StyledBox>
      <Sheet sx={{ width: "200px", height: "200px", background: "pink" }}>
        No elevation(shadow)
      </Sheet>
      <Sheet shadow="sm" sx={{ width: "200px", height: "200px" }} />
      <Sheet shadow="md" sx={{ width: "200px", height: "200px" }} />
      <Sheet shadow="lg" sx={{ width: "200px", height: "200px" }} />
    </StyledBox>
  );
}
```

### Variants

Sheets are available in 4 global variants: `solid` , `soft`, `outlined` and `plain`(default)

```jsx live
function SheetVariants() {
  return (
    <StyledBox>
      <Sheet variant="solid" sx={{ width: "200px", height: "200px" }} />
      <Sheet variant="outlined" sx={{ width: "200px", height: "200px" }} />
      <Sheet variant="soft" sx={{ width: "200px", height: "200px" }} />
      <Sheet
        shadow="lg"
        variant="plain"
        sx={{ width: "200px", height: "200px" }}
      />
    </StyledBox>
  );
}
```

### Colors

All components do support 6 color type: `primary`, `neutral`, `danger`, `info`, `success` and `warning`

```jsx live
function SheetVariants() {
  return (
    <StyledBox>
      <Sheet color="primary">HI There</Sheet>
      <Sheet color="neutral">HI There</Sheet>
      <Sheet color="info">HI There</Sheet>
      <Sheet color="danger">HI There</Sheet>
      <Sheet color="warning">HI There</Sheet>
    </StyledBox>
  );
}
```
