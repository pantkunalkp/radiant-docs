# Box

The Box component serves as a wrapper component for most of the CSS utility needs.

### The sx prop

```jsx live

function BoxSx(){
    return(
        <Box sx={{
            width: 300,
            height: 300,
            backgroundColor: "#357cf0"
        }}/>
    )
}
```
