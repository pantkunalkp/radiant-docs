# Checkbox

A checkbox, when clicked or tapped triggers an action or a series of actions.

Below are the relevant attributes for all checkbox types.

### Sizes

Checkbox are available in 3 sizes: `sm`, `md` (default) and `md`

```jsx live
function CheckboxSizes() {
  return (
    <Box sx={{ display: 'flex', flexDirection: 'column', gap: 2 }}>
      <Checkbox size="sm" label="sm Checkbox" />
      <Checkbox size="md" label="md Checkbox" />
      <Checkbox size="lg" label="lg Checkbox" />
    </Box>
  );
}
```

### States

Checkbox component only has three states: `checked`, `unchecked` and `indeterminate`.

```jsx live
function CheckboxStates() {
  return (
    <Box sx={{ display: 'flex', flexDirection: 'column', gap: 2 }}>
      <Checkbox checked={false} label="Default checkbox"/>
      <Checkbox checked={false} indeterminate label="Indeterminate checkbox"/>
      <Checkbox checked label="Checked checkbox"/>
      <Checkbox disabled label="Disabled checkbox" />
      <Checkbox disabled checked label="Disabled checked checkbox" />
      <Checkbox disabled indeterminate label="Disabled indeterminate checkbox" />
    </Box>
  );
}
```

### Colors

All components do support 2 color type: `primary`, `danger`

```jsx live
function CheckboxColor() {
  return (
    <StyledBox>
      <Checkbox color="primary" defaultChecked />
      <Checkbox color="neutral" defaultChecked />
      <Checkbox color="danger" defaultChecked />
      <Checkbox color="info" defaultChecked />
      <Checkbox color="success" defaultChecked />
      <Checkbox color="warning" defaultChecked />
    </StyledBox>
  );
}
```
